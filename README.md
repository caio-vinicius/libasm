# Libasm

## Getting started

Libasm requires [GNU Make v4.1+](https://www.gnu.org/software/make/), [GNU Compiler v7.5+](https://gcc.gnu.org/) and [NASM v2.13+](https://www.nasm.us/).

```bash
$ git clone https://github.com/caio-vinicius/libasm
$ cd libasm
$ make
```

## Usage

```bash
$ make test
$ ./a.out
```
